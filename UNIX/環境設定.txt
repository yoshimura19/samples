利用者のログイン時にシステムは/home下の.cshrcファイルの内容を実行した後、 .loginファイルの内容を実行するようになっている。
.alias  // コマンドなどの別名を定義
.emacs  emacsエディタ起動時の初期かファイル
.exrc  vi, exの初期かファイル
.login ログインしたとき実行するファイル名などを指定
.cshrc  csh起動時に初期化するパラメタを指定


環境変数
printenv  // 設定されている環境変数をいる
printenv 環境変数名 // 個々
setenv 環境変数名　定義内容　// 変更や追加定義
unsetenv 環境変数　　// 取り消す


which コマンド名  // コマンドの居場所を教えてくれるコマンド
which ruby
