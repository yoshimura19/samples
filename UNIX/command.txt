// ファイル名のパターンを検索
[指定法]grep [パラメータ][パターン] ファイル名
grep sendai ファイル名
(sendaiを含む行が表示)


// find
[指定法]find パス名[パラメータ]
find / -name core -print
ルート以下に（すべてのファイルが対象）coreという名前のファイルを探し、
見つかったらそのファイルのパス名を表示


リダイレクション機能（標準入出力の切り替え）
a.out > 出力ファイル名
ls > 出力ファイル名
// 追加書き込みは>>
// キーボードからの入力をファイルからの入力に切り替えることもできる
a.out < ファイル名


複数のコマンドを組み合わせることができる
パイプライン機能(標準入力と標準出力をつなぐ機能)
sort -t: /etc/passwd | lpr -Pipr


//コマンド名の別名登録
alias cat list   (catコマンドをlistと入力することができる)
